# [spark-cli](https://github.com/SPARKsaeedieckta/spark-cli)
*TG-CLI based broadcasting bot!*
# Install
```
cd $HOME
git clone https://github.com/SPARKsaeedieckta/spark-cli
cd spark-cli
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
Create a bot!
